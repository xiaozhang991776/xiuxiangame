# 仙途·轮回诀 — 修仙放置 / 养成游戏

纯前端（HTML / CSS / JavaScript）修仙主题放置游戏，**零外部依赖**。提供两种运行方式：

- **网页版**：浏览器直接打开 `index.html`
- **桌面版（Windows）**：双击 `仙途轮回诀.exe`（Electron 封装，存档跨启动保留）

---

## ✨ 核心特性

- **零依赖**：纯原生前端实现，无需构建、无需联网
- **全系统**：修炼 / 斗法 / 乾坤袋 / 三界历练 / 坊市 / 道途任务 / 道友 / 轮回 / 天赋 / 境界图鉴 / 仙路（大道+）/ 主线 / 灵宠 / 功法神通
- **数据持久化**：localStorage 自动存档 + 手动存档 + 导入导出 + 重置
- **响应式设计**：PC / 平板 / 手机自适应布局
- **离线收益**：关闭页面后按 50% 效率累积战力，最长 12 小时
- **墨色水墨风 UI** + 五行色系统 + 五品质装备体系
- **桌面版体验**：单实例锁、窗口居中、隐藏菜单栏，接近原生游戏

## 🚀 启动方式

### 方式 1：网页版（本地 HTTP 服务器，推荐）
```bash
cd xiuxian-game
python -m http.server 8080
# 浏览器打开 http://localhost:8080
```

### 方式 2：网页版（直接双击）
直接双击 `index.html` 即可（部分浏览器对 `file://` 协议有限制，推荐用方式 1）。

### 方式 3：桌面版（Windows）
双击 `dist/仙途轮回诀-win32-x64/仙途轮回诀.exe` 即可，无需服务器，存档跨启动保留。

## 🎮 玩法介绍

### 1. 角色创建
- **道号**：输入或掷签生成
- **灵根**：金 / 木 / 水 / 火 / 土 五行选择，影响属性克制
- **法相**：多种外观选择

### 2. 修炼系统
- 进入游戏后自动挂机修炼，战力每秒增长
- 修炼效率受 **境界 / 功法 / 装备 / 灵宠 / 轮回 / 天赋** 多层加成
- 战力满后可点击「突破」晋升小境界（9 层满后突破大境界）
- 大境界突破有成功率，可使用「破障丹」+20%
- **轮回门槛**：每世有「今生战力上线」（免费轮回门槛），达上限需轮回方可继续增长

### 3. 斗法（战斗）
- 境界递进的敌人与 Boss 可供挑战
- 回合制战斗：**五行克制**（金克木 → 木克土 → 土克水 → 水克火 → 火克金）
- 可学技能（连击 / 群伤 / 封印 / 混沌之力等）
- 法宝专属技能、灵宠专属技能

### 4. 乾坤袋（背包）
- 5 大分类：装备 / 丹药 / 材料 / 功法 / 法宝
- 装备支持：**穿戴**、**强化**（+1~+10）、**分解**（返还材料）
- 5 大品质：凡品 → 灵品 → 宝品 → 仙品 → 神品
- 9 种丹药，支持战斗中临时增益或永久属性

### 5. 三界历练（探索）
- 多场景：宗门 / 坊市 / 万妖林 / 灵矿洞 / 上古秘境 / 东海龙宫 / 混沌秘境
- 随机事件：奇遇 / 战斗 / 宝物 / 师尊指点 / 龙族考验
- 境界门槛限制高阶场景

### 6. 坊市（商店）
- 购买：装备、丹药、材料、功法、法宝、灵宠
- 出售：回收背包中多余物品换灵石
- 炼丹：用材料 + 灵石炼制丹药
- 含 **77 件拓展物品**

### 7. 道途任务 & 主线
- 递进任务链（战力累积 / 战胜敌人 / 历练探索 / 境界突破）
- 主线剧情章节，解锁奖励

### 8. 道友
- 论道比拼、排行榜（按综合战力评分）

### 9. 轮回 & 天赋
- **轮回**：免费散功重修（境界 + 战力归零）；**轮回草**保留境界与战力，仅重算寿元
- **天赋**：永久乘区，乘法叠加

### 10. 境界图鉴
- 一览全部境界（每层基础战力与满层突破成本）与每世轮回的战力上线
- 当前所处之境、当前世数对应的上限 **金底高亮**

### 11. 仙路·大道+（后期核心成长）
达 **大道** 境界后解锁 4 条**互不稀释**的独立乘区，叠加在 `基础 × 装备 × 灵宠 × 轮回 × 天赋` 之后：
- **法则领悟**：六条法则（剑 / 丹 / 阵 / 时空 / 因果 / 轮回）各 5 阶，每阶一个独立乘数
- **本命法宝**：5 种法宝、30 级，含独特战斗效果（吸血 / 反弹 / 护盾 / 先手 / 真伤）
- **洞天福地**：灵脉 / 药田 / 悟道崖 3 块 20 级，被动产出灵石 / 悟性经验，乘区独立
- **分身化身**：最多 3 个化身，按主修炼速率向战力池注水（不进战斗属性）

### 12. 灵宠（宠物）
- 多只灵宠，各有专属战斗技能

## 🔧 技术亮点

- **模块化 IIFE 架构**：无全局污染，模块解耦
- **数据驱动**：所有配置集中在 `config.js`，易扩展
- **响应式 CSS**：PC / 平板 / 手机三端适配
- **节流更新**：UI 每秒刷新一次，避免性能问题
- **异常处理**：存档失败、空数据、循环依赖、TDZ 等均有保护
- **桌面版**：单实例锁 + 窗口居中 + 隐藏菜单栏
