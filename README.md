# 仙途·轮回诀 — 修仙主题网页游戏

一款基于 HTML5 / CSS3 / JavaScript 实现的纯前端修仙主题网页游戏，无任何外部依赖，双击打开即可游玩。

## 🌟 核心特性

- **零依赖**：纯原生前端实现，浏览器直接打开 `index.html` 即可
- **8大系统完整**：角色、修炼、战斗、背包、探索、商店、UI、数据存储
- **数据持久化**：localStorage 自动存档 + 手动存档 + 导入导出 + 重置
- **响应式设计**：PC/平板/手机自适应布局
- **离线收益**：关闭页面后按 50% 效率累积修为，最长 12 小时
- **极致细节**：墨色水墨风 UI + 五行色系统 + 五品质装备体系

## 📁 文件结构

```
xiuxian-game/
├── index.html          # 主入口（UI骨架+脚本加载）
├── css/
│   └── style.css       # 仙侠风格主题（墨色水墨+金玉点缀）
└── js/
    ├── config.js       # 静态配置（境界/五行/技能/物品/敌人/地图）
    ├── save.js         # 存档系统（localStorage + 离线收益）
    ├── inventory.js    # 背包与装备（穿戴/强化/分解/炼丹）
    ├── cultivate.js    # 修炼与境界（突破/效率计算/属性）
    ├── combat.js       # 回合制战斗（技能/法宝/灵宠/五行克制）
    ├── explore.js      # 地图探索与任务（场景切换/随机事件）
    ├── shop.js         # 商店与交易（购买/出售/炼丹）
    ├── ui.js           # UI 渲染层（面板/弹窗/动效）
    └── main.js         # 主入口（加载/事件绑定）
```

## 🚀 启动方式

### 方式 1：本地 HTTP 服务器（推荐）
```bash
cd xiuxian-game
python -m http.server 8080
# 浏览器打开 http://localhost:8080
```

### 方式 2：直接双击
直接双击 `index.html` 即可（部分浏览器对 file:// 协议有限制，推荐用 HTTP 服务器方式）

## 🎮 玩法介绍

### 1. 角色创建
- **道号**：输入或掷签生成
- **灵根**：金/木/水/火/土五行选择，影响属性克制
- **法相**：4 种外观选择

### 2. 修炼系统
- 进入游戏后自动挂机修炼，修为每秒增长
- 修炼效率受**境界**、**功法**、**装备**三层加成
- 修为满后可点击「突破」按钮晋升小境界（9层满后突破大境界）
- 大境界突破有成功率，可使用「破障丹」+20%

### 3. 战斗系统（斗法）
- 11 个境界递进的敌人可供挑战
- 回合制战斗：**五行克制**（金克木，木克土，土克水，水克火，火克金）
- 12 个可学技能（含连击、群伤、封印、混沌之力等）
- 4 件法宝各有专属技能（护盾/镇压/反弹/万法）
- 4 个灵宠各有专属技能

### 4. 背包系统（乾坤袋）
- 5 大分类：装备 / 丹药 / 材料 / 功法 / 法宝
- 装备支持：**穿戴**、**强化**（+1~+10）、**分解**（返还材料）
- 5 大品质：凡品 → 灵品 → 宝品 → 仙品 → 神品
- 9 种丹药，支持战斗中临时增益或永久属性

### 5. 地图探索（三界历练）
- 7 个场景：宗门 / 坊市 / 万妖林 / 灵矿洞 / 上古秘境 / 东海龙宫 / 混沌秘境
- 22 种随机事件：奇遇 / 战斗 / 宝物 / 师尊指点 / 龙族考验
- 境界门槛限制高阶场景

### 6. 商店（坊市）
- 购买：装备、丹药、材料、功法、法宝、灵宠
- 出售：回收背包中多余物品换灵石
- 炼丹：用材料+灵石炼制丹药

### 7. 任务系统（道途任务）
- 10 个递进任务链
- 涵盖修为累积、战胜敌人、历练探索、境界突破

## 🎨 视觉设计

- **主色调**：墨色为底（#0d0a14），金玉为饰
- **五行色**：金 #d4af37 / 木 #6dbe6d / 水 #5dade2 / 火 #e74c3c / 土 #c4955a
- **品质色**：凡 #9ca3af / 灵 #4ade80 / 宝 #3b82f6 / 仙 #a855f7 / 神 #f43f5e
- **特殊效果**：突破光环、伤害飘字、五行克制高亮、灵气流转动画

## ⚙️ 技术亮点

- **模块化 IIFE 架构**：无全局污染，模块解耦
- **数据驱动**：所有配置集中在 `config.js`，易扩展
- **响应式 CSS**：PC/平板/手机三端适配
- **节流更新**：UI 每秒刷新一次，避免性能问题
- **异常处理**：存档失败、空数据、循环依赖等均有保护
- **键盘快捷键**：1-7 切换面板，B 突破，S 存档

## 🔧 数据存储

- 存档键：`xiuxian_save_v1`
- 自动存档：每 30 秒
- 离线收益：最长累积 12 小时，效率为在线 50%
- 导入导出：Base64 编码，跨设备同步
