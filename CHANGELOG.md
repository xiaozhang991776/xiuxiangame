## [1.1.0] - 2026-07-24

### 新增
- **桌面版（Electron 封装）**：提供 Windows 可执行文件 `dist/仙途轮回诀-win32-x64/仙途轮回诀.exe`，单实例锁、窗口居中、隐藏菜单栏，存档跨启动保留。
- **仙路·大道+ 后期成长体系**：达大道境界后解锁 4 条互不稀释的独立乘区（法则领悟 / 本命法宝 / 洞天福地 / 分身化身）。
- **游戏内中英切换**：设置面板新增「语言」下拉（中文 / English），玩家可随时自由切换；数据驱动翻译（境界名 / 五行 / 品质 / 核心 UI 文案 / 关键提示），零数据改动、零回归风险。

### 修复 & 调整
- 修复 `shop_ext.js` 因 TDZ（暂时性死区）导致 **77 件拓展物品丢失** 的严重 bug。
- 修复境界图鉴面板静默早返，增加防御层与异常暴露。
- 轮回草轮回改为 **保留境界与战力**（免费轮回仍归零散功）。
- 打赏弹窗收款码替换为游戏 logo；桌面版窗口图标去除收款码。
- **彻底移除「打赏」功能**（悬浮按钮 + 弹窗 + CSS + 教程引导指向），保留游戏内任务 / 敌人 / 剧情奖励机制。

### 文档
